## 1. Using terminal, connect to your server via SSH 
Replace the X's with your raspberry pi's IP address. Your password should be 'raspberry'

```
ssh pi@XXX.XXX.XX.X
```


## 2. Install PHP and restart apache
While connected via SSH, type the following commands followed by the enter key.

```
sudo apt-get install php -y
```
```
sudo service apache2 restart
```
